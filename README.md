# Chronic Kidney Disease (CKD) Prediction using Random Forest

A simple and accurate classification system to predict whether a patient has Chronic Kidney Disease (CKD) based on clinical features.

# Dataset

Name: Chronic Kidney Disease Dataset
Source: UCI Machine Learning Repository
Link: https://archive.ics.uci.edu/dataset/336/chronic+kidney+disease

# Requirements

Python 3.x
Required libraries:
    pandas
    scikit-learn
